OBSERVACOES:
=============


(*) O XMLNUKE JAVA fez uma implementação desse factory. Veja  se tem alguma
coisa interessante.

(**) Note que Context utiliza uma variável chamada _CONFIG que eh do tipo
NameValueCollection (uma coleção que recupera os itens por nome ao invés do
indice).  Basicamente essa classe processa as configurações que estão no
arquivo WEB.CONFIG. Veja como ficará no JAVA.

(***) Atenção para esse arquivo! Ele utiliza algumas definições existentes
só no .NET.  Talvez pudesse ser um pouco mais simplificada.
